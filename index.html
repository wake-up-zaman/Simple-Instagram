Conceptual Questions

Every browser has a JavaScript engine.The engine reads the JavaScript code and changes into machine executable instruction for the browser.The engine consists of two parts.Memory allocation happens in memory heap and call stack reads and executes the JavaScript code.JavaScript is interpreted language that means JS code is converted line by line instead of converting the whole program once.In JavaScript engine, the code faces various steps.Parse is the first stage of the engine.If the code has any syntactic error than it stops execution of the code.After checking all JS code in Parse, a data structure is created which name is Abstract Syntax Tree(AST).From the AST, JavaScript code converts into machine code.

JavaScript is a single threaded that means it executes everything by one thread.Whenever a single execution thread is done of executing whatever it was doing,it goes through a queue of events and executes them untill the queue is empty.It works as a loop that looks at all available events and execute them till there isn't anything left.This loop is called event loop. Basically the event queue is like a to-do list for the thread and the event loop is checking this to-do list consistently.If the queue is empty it goes to sleep but once an event gets into the queue it wakes up and does this thing all over again and again.
